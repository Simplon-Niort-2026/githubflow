# Dépôt d'entraînement GitHub Flow

1. Ouvre le projet lié à ce brief
2. Crée une *Issue* au format `nomprenom`, puis fais-en une *Pull Request*
3. Clone le projet et bascule sur ta branche
4. Crée un document au format `githubflow-nomprenom.md` contenant ton travail de recherche effectué précédemment
5. Commit et push le fichier
6. Ajoute ton binôme en *Reviewer*. Il relit, tu corriges si nécessaire, puis il valide
7. Ajoute ensuite ton formateur en *Reviewer*

Une fois toutes les étapes réalisées, tu partageras en solution :

* le lien vers la PR que tu as créée
* le lien vers la PR que tu as relue
